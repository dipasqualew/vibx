# Vibx

## Commands

- Use `bun run test` to run unit and integration tests across packages
- Use `bun run test:e2e` to run e2e tests
- Use `bun run build` to build all packages

## Definition of Done

Before any work is to be considered done, run the following commands and ensure they pass:

- `bun test:repeat`
- `bun test:e2e:repeat`
